# Roadmap

## Phase 0 — Foundation (Week 1)

- Initialize repo structure.
- Setup backend service skeleton.
- Setup web client shell.
- Define shared protocol (input/state messages).

## Phase 1 — First Playable (Weeks 2–4)

- Implement movement and docking.
- Implement basic market buy/sell loop.
- Add 4 locations, free-flight navigation, and approach-zone risk.
- Add pirate NPC encounter + simple combat.
- Persist player credits/cargo.

## Phase 2 — Multiplayer Hardening (Weeks 5–6)

- Improve network interpolation and lag handling.
- Add anti-cheat validations.
- Improve combat balancing.
- Add telemetry/logging for economy and death causes.

## Phase 3 — Expansion (Weeks 7+)

- Add more ship classes.
- Add mission contracts.
- Add player-vs-player risk zones.
- Begin multi-system design prep.

## Definition of Done (MVP)

A new player can complete a full trade run with realtime travel, survive/fail a pirate encounter, and observe persistent credits progression in a shared multiplayer session.
